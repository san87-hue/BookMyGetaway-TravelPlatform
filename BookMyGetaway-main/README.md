# BookMyGetaway – Group Travel Booking Platform

BookMyGetaway is a full-stack web platform that simplifies group travel planning by offering collaborative itineraries, shared expense tracking, and goal-based saving through Journey Jar. It helps users stay coordinated from planning to booking, ensuring a smooth and enjoyable travel experience.

##  Features
- Group Travel Hub – co-plan trips with shared itineraries
- Journey Jar – save gradually for upcoming trips
- Expense Tracking – fair cost splitting among members
- Secure bookings with smooth user flow
- Responsive UI for all devices

##  Tech Stack
**Frontend:** React.js  
**Backend:** Node.js + Express  
**Database:** MySQL  
**APIs:** RESTful services  
**Tools:** npm, webpack, Git, VS Code
